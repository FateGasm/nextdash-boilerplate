# Next.js dashboard boilerplate

---

A dashboard UI boilerplate built with **Next.js v16**.  
Features a login screen, a dashboard with a collapsible sidebar, and supports both light and dark themes.

## UI library used
- [shadcn/ui](https://ui.shadcn.com/docs/components) - UI components
- [lucide icons](https://lucide.dev/icons/) - Icons

## How-to
1. Clone the repo, then do initial setup:
```
$ npm install
```

2. Start the development server
```
$ npm run dev
```
The app will be running on http://localhost:3000 by default.

## Compatibility
Tested on Node.js v22.2.0.  
May not work with older versions.
